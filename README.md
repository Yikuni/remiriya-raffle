## 走马观花

这是一个抽奖箱插件，适合用于服务器，配合商店插件等插件一起使用。这是一个容易上手，方便使用的插件。

在这个插件中， 您可以创建任意个数的抽奖箱，在抽奖箱中添加多个物品，为每个物品分配不同的抽奖获得概率。可以为不同的抽奖箱设置不同的抽奖券(钥匙)。

## 插件版本

测试版本1.19.2

其它版本未经测试

## 详细介绍

### 指令介绍

1. 创建抽奖箱
2. 删除抽奖箱
3. 增加抽奖箱奖品
4. 删除抽奖箱奖品
5. 查询现有抽奖箱
6. 生成抽奖箱
7. op查询抽奖箱奖品
8. 帮助指令
